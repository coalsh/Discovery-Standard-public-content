This folder contains XML files for the MEDIN Discovery Metadata Standard that have been validated with the version of the Schematron that is associated with the Guidance 
notes version number in the filename e.g. MEDIN_Metadata_for_services_v3_1_1.xml would be an example discovery metadata file that has been validated by the Schematron that
checks the criteria for version 3.1.1 of the MEDIN Discovery Metadata Standard.
